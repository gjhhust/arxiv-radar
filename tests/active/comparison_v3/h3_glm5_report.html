An Image is Worth 32 Tokens for Reconstruction and Generation
📄 2406.07550 2026-03-14 22:51:33
📝 一句话总结
基于 VQ-VAE 2D 网格潜在表示，引入 Transformer 架构的 1D 序列 tokenization，实现 32 个 token 即可重建 256×256 图像且生成质量超越 DiT。
📖 中文摘要
本文提出 TiTok (Transformer-based 1-Dimensional Tokenizer)，将图像压缩为 1D 离散潜在序列，突破传统 2D 网格 tokenization 的冗余限制。在 ImageNet 256×256 上，仅用 32 个 token 即可重建高质量图像，gFID 达 1.97，较 MaskGIT 基线提升 4.21。在 512×512 分辨率下，TiTok 以 gFID 2.13 超越 DiT-XL/2 的 3.04，同时生成速度提升 410 倍，显著降低计算开销。

significant
💬 编辑点评
这篇论文建立在 VQ-VAE/VQGAN 的向量量化框架基础上，继承了 ViT 的 Transformer 架构设计，并借鉴了 MaskGIT 的非自回归生成范式。核心创新在于打破 2D 网格假设，将图像编码为 1D 潜在序列，配合两阶段训练策略（proxy code 预训练 + decoder 微调）。贡献实质上是对 image tokenization 范式的范式级重构——证明了 1D 表示不仅可行，而且在紧凑性和生成效率上显著优于传统 2D 方法。作者通过扎实的实验验证了「32 token 足以表征图像」这一反直觉发现，并展示了 scaling up tokenizer 可换取更紧凑的潜在空间。这是真正解决效率-质量权衡问题的创新工作，而非单纯调参刷点。

🔧 方法改进
vq-vae:1d-sequence
VQ-VAE 采用 2D 网格潜在表示，每张图需 256-4096 个 token；TiTok 改为 1D 序列，仅用 32 个 token 即可重建 256×256 图像，大幅压缩潜在空间。
vq-vae:proxy-code-training
VQ-VAE 直接回归像素值并依赖复杂损失函数；TiTok 引入两阶段训练，第一阶段用预训练 MaskGIT-VQGAN 的离散码作为 proxy code 训练，第二阶段微调 decoder，提升训练稳定性。
maskgit:compact-tokenizer
MaskGIT 使用 256-token 的 VQGAN tokenizer，生成需 8 步采样；TiTok 将 token 数压缩至 32，在相同生成框架下 gFID 从 6.18 提升至 2.77，推理速度提升约 2 倍。

基于 1D causal tokenization，引入 PCA-like 结构与 diffusion decoder，实现语义-频谱解耦与 SOTA 重建性能
📖 中文摘要
本文提出 SEMANTICIST，一种将可证明的 PCA-like 结构嵌入潜在 token 空间的新型视觉 tokenization 框架。现有视觉 tokenizer 主要优化重建保真度，但忽略了潜在空间的结构特性。本方法生成 1D 因果 token 序列，每个后续 token 以数学保证的递减解释方差贡献非重叠信息，类比于主成分分析。通过 diffusion decoder 解决语义-频谱耦合效应，实验显示该方法在 ImageNet 上达到 SOTA 重建 FID（0.72），较前 SOTA 提升近 10%。基于该方法训练的自回归模型仅需 32 tokens 即可达到与当前 SOTA 方法相当的性能，线性探测准确率最高达 63.5%。

💬 编辑点评
这篇论文建立在 VQ-VAE、TiTok 等 1D visual tokenizer 工作基础上，同时借鉴了 PCA 的降维思想与 nested dropout/Matryoshka representation 的训练策略。核心创新在于通过 nested classifier-free guidance 强制 token 序列形成 PCA-like 的递减方差结构，并用 diffusion decoder 解耦语义与频谱信息。实质贡献是首次在 visual tokenization 中引入可证明的结构性约束，使得 token 序列具有明确的语义层级，同时解决了此前 tokenizer 中语义-频谱耦合的问题。这个贡献具有实质价值：不仅重建性能提升显著，更重要的是为 visual representation learning 引入了新的结构化范式，对下游生成和理解任务都有明确益处。

significant

AliTok: Towards Sequence Modeling Alignment between Tokenizer and Autoregressive Model

📝 一句话总结
基于传统 VQ-VAE tokenizer，引入因果解码器约束编码器学习单向依赖的 token 序列，实现标准自回归模型在 ImageNet 上超越 SOTA 扩散模型
📖 中文摘要
本文提出 AliTok，解决图像 tokenizer 的双向依赖与自回归模型单向预测范式不匹配的问题。通过因果解码器约束双向编码器，强制 token 序列具有前向依赖性，同时引入 prefix tokens 和两阶段训练策略保持高重建质量。在 ImageNet-256 上，662M 参数模型达到 gFID 1.28，超越 SOTA 扩散模型 LightningDiT (1.35)，同时采样速度快 10×。在 ImageNet-512 上，318M 模型达到 gFID 1.39，刷新记录。
🏷️ 贡献类型
significant
💬 编辑点评
这篇论文建立在 VQ-VAE/VQ-GAN tokenizer 架构和 GPT 风格自回归生成范式的基础上。核心观察来自 LlamaGen 等工作的失败教训：raster-scan 自回归在图像上效果差，根本原因是 tokenizer 学到了双向依赖。方法的核心创新是用因果解码器作为正则化器，反向约束编码器的学习过程，而不是像 VAR、MAR 那样改变模型架构。这是一个简单但有效的对齐思路：不改模型，改数据（token 序列）的依赖结构。两阶段训练和 prefix tokens 是工程优化。实质贡献是提出并验证了一个重要假设——tokenizer 的依赖结构与生成模型范式的对齐比模型架构创新更关键。实验充分，消融清晰，首次让标准 decoder-only AR 超越扩散 SOTA，有方法论价值。但理论分析偏弱，为什么因果解码器约束能学到单向依赖，缺乏形式化或可视化解释（虽然提供了注意力图）。总体而言，这是一个有实质技术贡献且效果显著的工作。
