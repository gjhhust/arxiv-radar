本文提出 TiTok（Transformer-based 1-Dimensional Tokenizer），将图像从传统的 2D 网格 latent 表示转变为 1D 序列表示。通过将 256×256×3 图像压缩至仅 32 个离散 token（此前方法需要 256 或 1024 个 token），在 ImageNet 256×256 上使用相同 MaskGIT 生成框架达到 gFID=1.97，较基线提升 4.21。在 ImageNet 512×512 上，TiTok 超越 DiT-XL/2（gFID 2.74 vs 3.04），生成速度快 410 倍。最佳变体 gFID=2.13 超越 DiT-XL/2（3.04），且速度快 74 倍。

📖 中文摘要
本文提出AliTok解决自回归图像生成中tokenizer与自回归模型之间的根本性对齐问题。传统tokenizer为追求重建质量，使token之间形成双向依赖，与自回归模型的单向预测范式存在本质冲突。AliTok采用双向编码器结合因果解码器的架构，强制编码器产生具有前向依赖性的token序列。通过引入prefix tokens解决首行重建问题，并采用两阶段训练策略（第一阶段训练因果解码器约束的编码器，第二阶段冻结编码器并重新训练双向解码器提升重建质量），在ImageNet-256上，662M参数的模型达到gFID 1.28，超越SOTA扩散模型LightningDiT且推理速度快10倍。
